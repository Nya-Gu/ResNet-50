# ResNet-50 구현

## 1. ResNet-50을 구현하게 된 동기

ResNet은 여러 딥러닝 서적과 강의에서 반복적으로 등장하는 대표적인 모델로, 깊은 신경망의 학습을 가능하게 한 주요 전환점으로 평가받고 있습니다. 따라서 해당 모델을 직접 구현하며 구조를 이해해보고자 했습니다.

또한 이전에 구현한 VGGNet-16보다 ResNet-50이 훨씬 더 깊은 구조를 가지고 있는데, 그럼에도 불구하고 정말 ResNet-50이 안정적으로 학습되는지 확인하기 위해 ResNet-50을 구현하기로 했습니다.

## 2. 구현에 참고한 논문
**[1] Deep Residual Learning for Image Recognition**
([논문 링크](https://arxiv.org/abs/1512.03385))

위 논문을 참고해 ResNet-50을 구현했습니다. 논문에서 Shortcut Connection이 왜 효과 있었는가에 대한 저자들의 의견과 함께 다양한 실험 결과를 제시하고 있었습니다. 또 상당히 친절한 논문이었습니다. 사람들에게 연구 결과를 잘 전달하고 이를 설득시키기 위해 공들인 느낌을 받았습니다. 저같은 사람도 무난히 읽을 수 있게 작성해주셔서 감사했습니다.

추가로 논문을 읽으면서, 이러한 아이디어가 먼저 제안된 뒤 이를 검증하기 위한 실험이 수행된 것인지, 아니면 다양한 실험 결과를 바탕으로 이를 설명할 수 있는 구조와 논리가 정리된 것인지에 대한 궁금증이 들었습니다. 연구 과정이 궁금한 논문이었습니다.

## 6. 구현 및 모델 학습 후기
- **구현 난이도**  
ResNet-34와 달리 ResNet-50은 병목 구조를 가지고 있어 구현 난이도가 더 높았습니다. 특히 shortcut connection이 있어야 할 때와 그렇지 않을 때를 잘 구별해서 구현하는 것이 핵심이었으며, 구현 과정에서 논문을 반복적으로 참고하며 ResNet의 구조에 대한 이해를 더욱 명확히 할 수 있었습니다.

- **구현 정확도**  
Pytorch에 구현된 ResNet-50 모델과 비교하며 구현의 정확성을 점검했습니다. 예상보다 많은 차이를 발견할 수 있었고 아직 저의 구현 역량이 부족함을 수 있었습니다. 수정하는 과정에서 구현 디테일에 대한 이해를 보완할 수 있었습니다.

- **Shortcut Connection의 강력함**  
ResNet-50은 VGGNet-16보다 레이어가 3배 더 깊은 모델임에도 학습 과정에서 더 빠른 성능 향상을 보였고 학습에도 더 적은 시간이 소요되었습니다. 또한 파라미터가 VGGNet-16의 1/5 수준임에도 (138M vs 26M) 에 뒤지지 않는 성능을 보였다는 점에서 Shortcut connection의 강력한 효과를 체감할 수 있었습니다.

- **왜 VGGNet-16-bn이 ResNet-50보다 성능이 높았을까?**  
논문에 제시된 결과와 동일하게 ResNet-50이 VGGNet-16보다 더 높은 성능을 보였습니다. 하지만 VGGNet-16에 배치놈을 추가한 VGG16-bn보다는 성능이 높게 나오지 않았습니다. 여러 차례 반복 실험과 코드 검증을 진행했음에도 동일한 결과가 반복되었고, 결과적으로 이 문제의 원인에 대해 아래와 같이 추측하고 있습니다.

    - **오버피팅 가능성** : ViT 논문에 따르면 깊고 복잡한 모델일수록 오버피팅에 취약해 얕은 모델보다 성능이 낮게 나올 수 있다고 합니다. 저의 실험에서 VGG16-bn이 ResNet-50보다 나은 성능을 보인 것도 아마 이와 같은 이유가 아닐까 추측해봅니다. 학습에 사용한 데이터셋은 훈련 데이터가 1만개 정도로 작은 데이터셋이었고, 그래서 ResNet-50이 오버피팅에 취약해져 VGGNet-16보다 정확도가 낮게 나온 것으로 보입니다.

    - **모델 이상 무**: 모델 구현에 문제가 있었던 것인지 확인하기 위해 Pytorch에 구현된 ResNet-50을 학습시켜봤지만 전과 동일하게 VGG16-bn보다 성능이 낮게 나왔습니다. 따라서 코드나 모델의 이상보다는 작은 데이터셋으로 인한 오버피팅으로 인해 모델 성능이 VGGNet-16보다 낫게 나온 것으로 이해하고 있습니다.
    
    - **차후 실험 방향**: 데이터 증강에 대해 좀 더 공부한 뒤 다시 성능 비교 실험을 하려고 합니다. 또한 사전 훈련된 ResNet-50과 VGG16-bn을 파인튜닝했을 때 두 모델 중 어느 쪽이 성능이 더 높게 나오는지 시험하려고 합니다. (구현6 레포지토리 참고바랍니다.)
